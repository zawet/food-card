# 美食抽卡小程序

中山美食打卡抽卡小程序，包含美食卡片抽取、全部卡片浏览、抽卡记录等功能。

## 功能

- 美食抽卡 - 随机抽取中山美食卡片
- 网红打卡 - 专门抽取网红打卡点美食
- 全部卡片 - 查看所有美食卡片
- 抽卡记录 - 查看历史抽卡记录
- 饮品数据 - 包含100款热门饮品数据

## 技术栈

- HTML5 + CSS3 + JavaScript
- 无需后端，纯前端项目
- 数据存储在 localStorage

## 运行

直接在浏览器中打开 index.html 即可运行。

或使用本地服务器：

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 数据

- foods.json - 中山美食卡片数据（180张卡片）
- drinks.json - 饮品卡片数据（100款饮品）
